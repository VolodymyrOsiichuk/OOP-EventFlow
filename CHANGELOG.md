# [1.0.0] - 2026-06

## Added

### Ітерація 1

- Багатошарова архітектура.
- Доменні сутності:
  - Event
  - Participant
  - Registration
  - Venue

- Repository Pattern.
- Factory Pattern.
- Створення подій.
- Реєстрація учасників.
- Unit-тести.
- GitHub Actions CI.

### Ітерація 2

- JSON persistence.
- EventPersistenceService.
- EventMapper.
- EventQueryService.
- Скасування реєстрації.
- LINQ-запити.
- Статистика за категоріями.
- Розширене консольне меню.

### Ітерація 3

- Інтеграційні тести.
- Fault handling.
- Coverage через Coverlet.
- Test Strategy.
- Test Matrix.
- Quality Gate.
- Автоматична перевірка тестів у CI.

### Ітерація 4

- USER_GUIDE.
- DEVELOPER_GUIDE.
- DEMO сценарій.
- FINAL_REPORT.
- Матриця покриття курсу.
- Підготовка релізу v1.0.0.
