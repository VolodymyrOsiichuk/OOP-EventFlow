# USER GUIDE

## Призначення

EventFlow — консольний застосунок для управління подіями та учасниками.

Система дозволяє створювати події, реєструвати учасників, скасовувати реєстрації та зберігати дані між запусками програми.

---

# Запуск програми

Виконайте команду:

```bash
dotnet run --project src/EventFlow.Console
```

Після запуску відкриється головне меню.

---

# Головне меню

```text
1. Create Event
2. Register Participant
3. Cancel Registration
4. View All Events
5. Active Events
6. Statistics
7. Save Data
0. Exit
```

---

# Створення події

Оберіть пункт:

```text
1. Create Event
```

Введіть необхідні дані:

- назву події;
- дату;
- місткість;
- місце проведення;
- категорію.

Після успішного створення система повідомить про результат.

---

# Реєстрація учасника

Оберіть:

```text
2. Register Participant
```

Вкажіть:

- ідентифікатор події (Event Id);
- ім'я учасника;
- email.

Система перевіряє:

- наявність вільних місць;
- відсутність дубліката;
- актуальність події.

---

# Скасування реєстрації

Оберіть:

```text
3. Cancel Registration
```

Вкажіть:

- Event Id;
- Email учасника.

Після успішного виконання місце звільняється.

---

# Перегляд подій

Оберіть:

```text
4. View All Events
```

Для кожної події відображаються:

- Id;
- назва;
- категорія;
- кількість учасників;
- доступні місця.

---

# Активні події

Оберіть:

```text
5. Active Events
```

Відображаються лише майбутні події.

---

# Статистика

Оберіть:

```text
6. Statistics
```

Система покаже кількість подій у кожній категорії.

---

# Збереження даних

Оберіть:

```text
7. Save Data
```

Дані буде збережено у файл JSON.

---

# Автоматичне збереження

Під час завершення роботи програми дані автоматично зберігаються.

---

# Відновлення даних

При запуску застосунку система автоматично намагається завантажити дані з файлу.

Якщо файл відсутній або пошкоджений, програма продовжить роботу без аварійного завершення.

---

# Типові помилки

## Event is full

Усі місця на подію вже зайняті.

## Participant already registered

Користувач уже зареєстрований на цю подію.

## Participant is not registered

Спроба скасувати неіснуючу реєстрацію.

## Registration is closed

Подія вже розпочалася або завершилася.

---

# Завершення роботи

Для завершення роботи використайте:

```text
0. Exit
```

Перед виходом виконується автоматичне збереження даних.
