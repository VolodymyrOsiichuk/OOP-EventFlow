# Release Plan v1.0.0

## Мета релізу

Версія 1.0.0 є фінальним навчальним релізом проєкту EventFlow та демонструє застосування принципів ООП, SOLID, патернів проєктування, persistence, тестування та контролю якості.

---

## Що входить до v1.0.0

### Функціональність

- створення подій;
- реєстрація учасників;
- скасування реєстрації;
- перегляд подій;
- пошук і фільтрація подій;
- статистика за категоріями.

### Persistence

- збереження у JSON;
- завантаження даних із JSON;
- відновлення стану після запуску.

### Якість

- unit tests;
- integration tests;
- fault handling;
- code coverage;
- GitHub Actions CI.

### Документація

- README;
- USER_GUIDE;
- DEVELOPER_GUIDE;
- TESTING;
- CHANGELOG;
- FINAL_REPORT.

---

## Функції після завершення курсу

Не входять до версії 1.0.0:

- база даних;
- веб-інтерфейс;
- автентифікація користувачів;
- логування через сторонні бібліотеки;
- REST API.

---

## Допустимий технічний борг

### Консольний інтерфейс

Інтерфейс мінімалістичний і призначений лише для демонстрації функціональності.

### Зберігання у файлах

Для навчального проєкту використовується JSON замість повноцінної СУБД.

### Dependency Injection

Залежності створюються вручну в Program.cs.
